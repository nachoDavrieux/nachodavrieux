Juego 2048

Se trata de crear el juego 2048 utilizando el lenguaje de programación C y C++, tarea para la universidad.
El 2048 es un juego de rompecabezas numérico en el que el objetivo es combinar fichas con números para alcanzar el número 2048 en una cuadrícula de 4x4. Las fichas se combinan deslizándolas en direcciones específicas y fusionando fichas adyacentes del mismo valor. El juego continúa hasta que ya no sea posible realizar movimientos válidos.
